GDFC — Three Localized Safety Layers
🔍
On-Edge Sensor Voting Logic idle
Local ML classifier cross-checks redundant sensor signals in real time. On anomaly (pitot icing), it isolates the corrupted source using GPS-velocity fallback — no cloud round-trip required.
🛡️
Graceful Flight Degradation idle
Instead of a full autopilot drop-off, the system enters Safe-Hold Mode — stabilising wing-level trajectory and constraining pitch to safe envelopes while the pilot re-orients.
🖥️
UI De-cluttering & Cognitive Assist idle
Secondary error codes are suppressed. A single high-visibility recovery protocol is surfaced, giving the pilot a 15-second buffer to absorb the transition before resuming manual input.
