GDFC — Three Localized Layers

01
On-Edge Sensor Voting Logic idle
Local ML classifier cross-checks redundant sensor signals in real time. On anomaly (e.g. icing), it isolates the corrupted source using non-corrupted data — no cloud round-trip.
02
Graceful Flight Degradation idle
Instead of a full autopilot drop-off, the system enters Safe-Hold Mode — stabilizing wing-level trajectory and holding safe pitch envelopes while the pilot re-orients.
03
UI De-cluttering & Assistive Interface idle
Secondary error codes are stripped from the dashboard. A single high-visibility recovery protocol is shown, giving the pilot a ~15s buffer before resuming manual input.
